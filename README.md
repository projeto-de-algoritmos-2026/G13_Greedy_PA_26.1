# G13 Greedy PA 26.1

## Objetivo

Projeto desenvolvido para a disciplina de Projeto de Algoritmos (PA), utilizando algoritmos gulosos (Greedy) para resolução de problemas propostos na matéria.


## Alunos
- Amanda Gonçalves Sobrinho Abreu (211030925)
- Arthur Rodrigues Sousa (211030291)
  
## Funcionamento

A aplicação possui interface web desenvolvida com React e TypeScript, permitindo executar e visualizar o funcionamento dos algoritmos implementados.

O projeto utiliza Vite para desenvolvimento frontend e Bun como gerenciador de pacotes/runtime.



##  Pré-requisitos

Antes de executar o projeto, é necessário ter instalado:

- Node.js 20+
- Bun
- Git

### Instalar dependências

```bash
bun install
```
```bash
bun run dev
```
## Tecnologias Utilizadas
- React
- TypeScript
- Vite
- Bun
